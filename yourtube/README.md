# 🎬 YourTube - YouTube Clone

YourTube is a full-stack YouTube clone web application that allows users to watch, upload, and interact with videos just like YouTube.

---

## 🚀 Features

* 🔐 User Authentication (Login / Signup)
* 📹 Video Upload & Streaming
* 👍 Like / Dislike System
* 💬 Comment System (with translation support)
* 📺 Channel Creation
* 🔍 Search Videos
* 🌐 Responsive Design

---

## 🛠️ Tech Stack

### Frontend

* React.js
* Tailwind CSS

### Backend

* Node.js
* Express.js

### Database

* MongoDB

---

## 📂 Project Structure

```text
client/     → Frontend (React)
server/     → Backend (Node + Express)
```

---

## ⚙️ Installation

### 1. Clone the repository

```bash
git clone https://github.com/chinnu554/yourtube.git
cd yourtube
```

### 2. Setup Backend

```bash
cd server
npm install
npm start
```

### 3. Setup Frontend

```bash
cd client
npm install
npm start
```

---

## 🔑 Environment Variables

Create a .env file inside server folder:

```env
MONGO_URI=your_mongodb_url
JWT_SECRET=your_secret_key
```
